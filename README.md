# Omango-Futex-RS

This is implement futex for Rust .<br />

## Table of Contents

- [Usage](#usage)
- [Compatibility](#compatibility)
- [License](#license)

## Usage

Add this to your `Cargo.toml`:
```toml
[dependencies]
omango-futex-rs = "0.1.2"
```

## Compatibility

The minimum supported Rust version is 1.49.
